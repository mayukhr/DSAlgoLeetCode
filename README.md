# Data Structures Repo

This repo contains DS Algo and LeetCode-style practice in JavaScript and TypeScript, mainly for interview prep.

## Contents

1. LeetCode-style problems
2. General algorithm exercises
3. Data structure drills

## Layout

Problems live under `problems/<topic>/<problem-slug>/`:

- `solution.ts` or `solution.js` — your implementation (named exports).
- `solution.test.ts` or `solution.test.js` — tests (often generated by an agent; see [AGENTS.md](AGENTS.md)).

Example: [problems/example/two-sum/](problems/example/two-sum/). The example `solution.ts` starts as a stub; implement it to make the tests pass.

## Scripts

```bash
npm install
npm test          # run all tests once
npm run test:watch  # watch mode while you code
```

### Run one test file

Pass a path (or pattern) to Vitest. It matches file paths, not only test names.

```bash
# single file, once
npx vitest run problems/example/two-sum/solution.test.ts

# same file in watch mode
npx vitest problems/example/two-sum/solution.test.ts

# optional: filter by test name inside that file
npx vitest run problems/example/two-sum/solution.test.ts -t "duplicate"
```

Paths are relative to the repo root.

## Agent workflow

Agents should add or update tests beside your solution and include corner cases. Details: [AGENTS.md](AGENTS.md).
